# 📂 訓練資料集目錄 (Dataset Directory)

本資料夾存放用於手勢/方向識別模型訓練與驗證的灰階影像資料。

---

## 📂 資料夾結構

```text
dataset/
├── train/
│   ├── up/             # 向上箭頭/手勢圖片
│   ├── down/           # 向下箭頭/手勢圖片
│   ├── left/           # 向左箭頭/手勢圖片
│   ├── right/          # 向右箭頭/手勢圖片
│   ├── null/           # 背景/無手勢圖片
│   └── info.labels     # 類別標籤設定檔
├── validation/         # 獨立的驗證集 (可選)
└── _old/               # 封存的舊版訓練資料
```

---

## ⚙️ 訓練時的資料處理規則
在 `train_and_quantize.py` 訓練流程中，程式會執行以下操作：
1. **載入目錄**：讀取 `dataset/train/` 底下的影像。
2. **動態分割**：設定 `validation_split=0.2`（80% 用於訓練，20% 用於驗證）。
3. **色彩模式**：強制轉換為 **`grayscale` (灰階)**。
4. **影像縮放**：強制縮放為 **`96x96` 像素**。
5. **數值歸一化**：將影像畫素除以 255.0，縮放至 `[0.0, 1.0]` 的浮點數。

---

## 📥 如何擴充訓練資料？
若您想增加新樣本以提升模型的精準度：
1. 將拍攝的照片放入 `dataset/train/` 底下對應的類別資料夾（如 `up/`、`down/` 等）。
2. 照片檔案格式支援 `.jpg`, `.jpeg`, `.png`, `.bmp`。
3. 建議照片背景儘量維持單純，或者在訓練前先將手部區域裁剪出來，以獲得最佳的學習效果。
4. 放好照片後重新運行 `python train_and_quantize.py` 進行訓練。
