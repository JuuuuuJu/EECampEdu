# Keras Source Models

This folder keeps deploy source models in `.keras` format.

Current source models:

```text
Baseline_CNN.keras
Mini_ResNet.keras
MobileNetV1_0.25.keras
MobileNetV2_0.35.keras
Separable_CNN.keras
```

The firmware deploy pipeline does not use the `.tflite` files generated by the
training script directly. Instead, it reads `.keras` and performs calibration
quantization in:

```text
firmware/pc/tools/quantize_keras_model.py
```

Generated `.tflite` files should be treated as local outputs and can be
regenerated.
