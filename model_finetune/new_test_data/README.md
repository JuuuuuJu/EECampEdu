 

# 📂 實測資料集目錄 (Real-World Test Data Directory)

本資料夾存放一組獨立的、在真實生活場景中拍攝的方向手勢圖片（`new_test_data/`）。

---

## 🎯 目的與用途

* **泛化能力測試**：此資料集**不參與模型的訓練與常規驗證**。它作為獨立的「測試集（Test Set）」，專門用來評估模型在未見過的環境、光線與相機鏡頭下的表現（泛化能力）。
* **實測準確度 (Real Acc)**：在 `train_and_quantize.py` 與評估腳本中運算出的 `Real Acc` 指標，就是利用本資料夾下的圖片進行推算。

---

## 📂 資料夾結構

```text
new_test_data/
├── up/             # 向上箭頭/手勢實測圖
├── down/           # 向下箭頭/手勢實測圖
├── left/           # 向左箭頭/手勢實測圖
├── right/          # 向右箭頭/手勢實測圖
├── null/           # 背景無手勢實測圖
├── info.labels     # 類別標籤設定檔
└── _old/           # 封存的舊版實測資料
```

---

## 🔍 前處理流程

當評估腳本讀取此資料集時，會將影像：

1. 轉換成灰階。
2. 縮放為 `96x96`。
3. 像素歸一化至 `[0.0, 1.0]`，再依 TFLite 模型輸入層之 scale 與 zero_point 量化為 `int8` 送入預測。
